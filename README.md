# AI-Powered Customer Support System

## Overview
This project implements an AI-powered customer support system using a multi-agent architecture.
A router agent analyzes incoming queries and delegates them to specialized agents:
Support Agent, Order Agent, and Billing Agent.

## Tech Stack
- Backend: Hono (TypeScript)
- Frontend: React (Vite)
- Database: PostgreSQL
- ORM: Prisma
- AI: Vercel AI SDK
- Infrastructure: Docker
